# 22mic7238 - Backend Evaluation

Notification system and vehicle maintenance scheduler with priority-based processing and automated scheduling.

## Quick Start

```bash
cd notification_app_be
npm install
npm start

cd vehicle_maintenance_scheduler
npm install
npm start
```

## Projects

- **notification_app_be** - Notification processing system
- **vehicle_maintenance_scheduler** - Vehicle scheduling with priority algorithm
- **logging_middleware** - Logging infrastructure

## Requirements

- Node.js 18+
- Environment variables configured in `.env` files
